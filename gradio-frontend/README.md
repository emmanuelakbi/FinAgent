# FinAgent — AI Trading Signals

FinAgent is an AI-powered trading signal generator that analyzes a watchlist of tickers and produces structured BUY / SELL / HOLD recommendations with confidence scores, entry, stop-loss, and target prices.

Under the hood, a multi-agent [CrewAI](https://github.com/joaomdmoura/crewAI) pipeline coordinates research, technical analysis, and risk agents, backed by a [vLLM](https://github.com/vllm-project/vllm) inference endpoint. The Gradio frontend presents the workflow as a dark financial terminal dashboard: configure risk tolerance, portfolio value, and trading style, then watch the live agent activity feed stream intermediate outputs while signal cards render as each ticker completes.

## Configuration

This Space requires the following environment variable:

- `VLLM_ENDPOINT_URL` — URL of the vLLM OpenAI-compatible inference endpoint used by the agent pipeline.

## Usage

1. Enter a comma-separated watchlist (up to 10 tickers), for example: `AAPL, NVDA, TSLA, BTC-USD`.
2. Select a risk tolerance, portfolio value, and trading style.
3. Click **Analyze** and watch the activity feed. Signal cards appear as each ticker finishes.

## Disclaimer

Signals are generated by AI agents for informational and educational purposes only and do not constitute financial advice. Always do your own research before trading.
